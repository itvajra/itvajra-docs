# Approvals (Policy Approvers)

## View pending approvals

1) Open `Approvals` from the sidebar.
2) Review the list of pending requests.
3) Open a request to review policy details.

## Approve or request changes

1) Open the approval request.
2) Review the policy content, notes, and metadata.
3) Choose `Approve` to finalize or `Request Changes` to send it back.

## Track decisions

- Approved policies move to an approved state and become available for export.
- Requests for changes return to authors/admins for updates.
