# Policies (Admins and Authors)

## Create a policy

1) Open `Policies`.
2) Select `New Policy`.
3) Choose a template and fill in required fields.
4) Review the generated content and edit if needed.
5) Save the policy as a draft.

## Submit for approval

1) Open a draft policy.
2) Select `Submit for Review`.
3) Add notes if required.
4) The policy moves into the approval queue.

## Edit or update a policy

1) Open the policy.
2) Select `Edit` and apply changes.
3) Save to create a new version.

## Archive or restore

- Use `Archive` to remove outdated policies from active lists.
- Use `Restore` to bring a policy back to active status.
