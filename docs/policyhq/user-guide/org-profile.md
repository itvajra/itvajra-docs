# Org Profile (Admins)

Org Profile controls organization-wide settings and metadata used by PolicyHQ.

## General information

- Organization name
- Organization size
- Industry
- Policy tone (Formal or Plain)

## Technical stack

- Cloud provider
- Identity provider
- Endpoint management
- Security tools and related fields (as applicable)

## Platform features

- Require acknowledgements: when enabled, users must acknowledge assigned policies
- Enable review workflow: when enabled, policies require approval before publication

## Compliance and approvals

- Default approvers (if configured)
- Review cadence settings

## Save changes

1) Update the desired fields.
2) Select `Save`.
3) Confirm the success message.
