# Templates (Admins)

## View templates

1) Open `Templates`.
2) Review available templates and their frameworks.

## Create or edit a template

1) Select `New Template` or open an existing template.
2) Update title, framework, and mapped controls.
3) Edit the template content and variables.
4) Save changes.

## When to update templates

- When frameworks change or new controls are required.
- When policy language needs revision.
