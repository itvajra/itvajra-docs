# Drafts (Authors)

Drafts are policies you are actively working on but have not submitted for approval.

## Open drafts

1) Open `Drafts` from the sidebar.
2) Select a draft to continue editing.

## Edit a draft

1) Update content and metadata.
2) Save changes to create a new version.

## Submit a draft

1) Open the draft.
2) Select `Submit for Review`.
3) Add a comment if required.

## Common reasons to keep a draft

- Waiting on stakeholder input
- Reviewing policy tone or framework alignment
- Coordinating with business unit owners
