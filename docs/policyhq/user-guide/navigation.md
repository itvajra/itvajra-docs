# Navigation Basics

## Main layout

- Sidebar: primary navigation across modules
- Header: page context and user menu
- Page header: page title, filters, and primary actions

## Common navigation items

- My Policies / Policies to Acknowledge
- Policies (admin/author list)
- Approvals
- Templates
- Reports
- Exports
- Audit Logs
- Business Units
- Users and Imports

## Tips

- Use search and filters to reduce large lists.
- Open a policy from any list to view details and actions.
