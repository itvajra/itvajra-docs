# Quickstart: Org Admin

Use this checklist to get your organization running quickly.

## First-time setup

1) Open `Org Profile` and complete the organization details.
2) Verify `Require acknowledgements` and `Enable review workflow` settings.
3) Save changes.

## Core tasks

- Create business units and assign owners.
- Import users and assign roles/business units.
- Review templates and create initial policies.
- Submit policies for approval and publish.

## Daily workflow

- Review policy status and due dates.
- Monitor approval queue (if you are also an approver).
- Review reports and audit logs.
